# DevOps for Developers Project 76

https://climbingchevalier.ru/

## Requirements

- SSH-доступ к серверам

## Install Ansible dependencies

Установить зависимости:

```bash
make install
```

## Prepare servers

Запустить подготовку серверов:

```bash
make prepare
```

## Deploy Redmine

Запустить Redmine на серверах:

```bash
make deploy
```
